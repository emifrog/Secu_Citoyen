Détecteurs de fumée

Pourquoi c'est important

Les détecteurs de fumée sont essentiels pour alerter rapidement en cas de début d'incendie, même pendant le sommeil.

Conseils d'entretien

  • Testez mensuellement avec le bouton de test
  • Remplacez les piles annuellement
  • Dépoussiérez régulièrement
  • Remplacez le détecteur tous les 10 ans

Soyez attentif aux signaux sonores (bips) indiquant une pile faible.

fire_extinguisher

Extincteur

Types d'extincteurs

  • Type A : Feux de matières solides
  • Type B : Feux de liquides
  • Type C : Feux de gaz

Entretien

  • Vérifiez la pression (zone verte)
  • Révision professionnelle tous les 5 ans

Familiarisez-vous avec le mode d'emploi et envisagez une formation pratique pour une utilisation efficace en cas d'urgence.

Plan d'évacuation

Points essentiels

  • Identifier toutes les sorties possibles
  • Définir un point de rassemblement
  • Prévoir des alternatives si une sortie est bloquée

Organisez des exercices d'évacuation réguliers avec tous les membres du foyer.

Conseils de stockage

  • Endroit frais et sec
  • Bonne ventilation
  • Contenants d'origine hermétiques
  • Éloigné des sources de chaleur
  • Hors de portée des enfants
